# 📊 Dashboard Djezzy

> Final Year Project (PFE) – Intelligent Customer Insight & Sentiment Analysis Prediction Platform for Djezzy Telecom 🇩🇿

A full-stack web dashboard built with Django that integrates AI models to analyze customer satisfaction, predict churn, and assist with customer sentiment analysis, all from a clean and intuitive UI.

---

## 🚀 Features

- 📈 **Client Satisfaction Dashboard**: Visualizes NPS & CSAT across services.
- 💬 **AI-Powered Comment Classification**: Uses a fine-tuned CamemBERT model to analyze client feedback in French.
- 🧠 **Churn Prediction Module**: Smart insights on potential client losses.
- 🔎 **Sentiment Analysis**: Classifies customer mood (positive, neutral, negative).
- 📂 **Admin Panel**: Manage models, user permissions, and exports.
- 🌍 **Multilingual Ready**: Designed for Algerian customer context.
